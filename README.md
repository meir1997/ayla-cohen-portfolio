# אילה כהן - פורטפוליו אתר

אתר פורטפוליו מודרני ויוקרתי לאילה כהן - עיצובת וארכיטקטלית.

## 🚀 טכנולוגיות

- **Next.js 14** - React framework עדכני
- **Tailwind CSS** - סטיילינג עדכני וקל
- **TypeScript** - בדיקות סוגים חזקות
- **RTL Support** - תמיכה מלאה לעברית

## 📁 מבנה הפרויקט

```
src/
  app/
    page.tsx           # דף הבית
    layout.tsx         # ה-layout הראשי
    globals.css        # סטיילים גלובליים
    projects/
      page.tsx         # עמוד הפרויקטים
    about/
      page.tsx         # עמוד אודות
    contact/
      page.tsx         # עמוד יצירת קשר
  components/
    Header.tsx         # Header עם נווגציה
    Footer.tsx         # Footer
  lib/
    projects.ts        # נתוני הפרויקטים
portfolio_images/
  numbered/            # כל התמונות ממוסדרות לפי פרויקט
    מבוא חורון/        # 37 תמונות
    פנטאוז בירושלים/   # 28 תמונות
    סלון יוקרתי/       # 5 תמונות
    מטבח יוקרתי/       # 14 תמונות
```
