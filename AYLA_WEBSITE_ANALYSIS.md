# אתר של אילה כהן - ניתוח עיצוב וארכיטקטורה

## 📐 עיקרונות העיצוב הנוכחיים של אילה

### 1. **סגנון כללי**
- **מינימליסטי + אלגנטי** - פחות הוא יותר
- **לבן וקרים כצבע בסיסי** - פלטה צבעים מוגבלת ופשוטה
- **טיפוגרפיה אלגנטית** - לוגו בכתב יד (AC)
- **הרבה whitespace** - נושימה בתוכן

### 2. **מבנה העמודים**

#### דף הבית:
1. **Header** - לוגו + navigation (STUDIO | SHOP)
2. **Hero Section** - תמונה גדולה של פרויקט יוקרה
3. **טקסט תיאור** - על השירותים בצד של התמונה
4. **CTA Button** - קריאה לפעולה ברורה

#### דף Studio/Portfolio:
1. **Header with profile** - תמונה של אילה + תיאור אישי
2. **About section** - אילה כהן, אדריכלות ועיצוב פנים
3. **Portfolio grid** - תמונות של פרויקטים (בעיקר דירות יוקרה)
4. **Services section** - 4 שירותים עיקריים

### 3. **סרטון + טקסטוריות**
- **תמונות בעלות גבוהה מאד** - פוטוגרפיה מקצועית של spaces
- **רקע חלק וקל**
- **טקסטורה ויזואלית subtle** (גלגלים, קווים עדינים)

### 4. **פלטת צבעים**
```
Primary: White (#FFFFFF)
Secondary: Cream/Beige (#F5F5F0, #EFEFEA)
Accents: Dark Gray/Black for text (#2A2A2A)
Gold/Bronze touches for luxury
Border/Frame: Pink/Coral subtle edge
```

### 5. **שירותים המוצגים**
1. **Home Styling** (סטילינג בית)
2. **Interior Design** (עיצוב פנים)
3. **Architectural Planning** (תכנון אדריכלי)
4. **Building Permits** (היתרי בנייה)
